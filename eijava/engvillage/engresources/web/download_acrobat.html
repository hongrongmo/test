Adobe Reader 5.0 Settings
  1. Choose Edit  >  Preferences  >  General option.
  2. Select the Options menu in the left pane.
  3. Ensure that the following Web Browser Options have a check mark:
    • Display PDF in Browser
    • Check Browser Settings when starting Acrobat.
    • Allow Fast Web view
  4. Ensure that the following Web Browser Option has been unchecked:
    • Allow Background Downloading

Adobe Reader 6.0 Settings
  1. Choose Edit  >  Preferences
  2. Select the Internet menu in the left pane.
  3. Ensure that the following Web Browser Options have a check mark:
    • Display PDF in Browser
    • Check browser settings when starting Reader
    • Allow Fast Web view
  4. Ensure that the following Web Browser Option has been unchecked:
    • Allow speculative downloading in the background.
  5. Locate the Connection Speed drop-down located in the Internet Options section. Select the highest possible Connection Speed preferably LAN.
