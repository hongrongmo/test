Mapping tool

The Mapping tool is standard in GEOBASE and GeoRef databases.  It is about graphically showing, or mapping the significant geographic features that are mentioned in the result set of a search. After running a search, users can choose the "Show Map" button and a map of their results will appear.  The map plots different geographical data from GEOBASE and GeoRef search results on a digital map utilizing Google map technology.

 

There is also an additional facet included with the mapping tool, called "Geographic Terms".  This is not to be confused with the "Country" facet.  The Geographic Term represents the geographic locations that are plotted on the map.  The Country represents the country in which the Author Affiliation is located.

 

Icons on the map represent geographic terms – roll over icons to display term and record count.

 

Zoom in and out ; Navigate with arrows or by panning over map.

 

Clicking on map icon generates the subset of search results which include that feature/location (same as selecting term from facets and clicking “include”).  Once the new set of search results appears, the map will close.  Clicking the "Show Map" button again will bring up the map again.

Users can click the "Hide Map" button again to hide the map.
