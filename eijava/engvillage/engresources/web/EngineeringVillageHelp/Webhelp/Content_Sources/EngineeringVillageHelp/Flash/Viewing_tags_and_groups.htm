Viewing Tags and Groups

You can view tags from the Tags + Groups tab.

When clicking on this tab the following window appears:

Search Tags: you can limit your search to search tags in the public, private or “My institution” repositories. For example, selecting public will perform the search on all publicly made tags.

Type the key word / tag you are looking for. A drop down menu will show up with tags containing the letters that you are typing. You can select a tag from the list or continue to type the term/s that you are looking for.

The tag cloud in the center of the page maps the existing tags in the public, private or “my institution” depositories. Large and bold words indicate the existence of a large number of records tagged with them.

You can click any of the terms and review the records tagged with them.

Rename Tags: use rename tags to edit the tags you assigned.

Delete Tags: use delete tags to remove tags you assigned

View/Edit groups: use view/edit groups to create new group, add or remove email address from groups you created or review groups you created.
