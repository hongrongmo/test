Creating Groups

To create a group click on View/Edit groups.

Click on Create a New Group (+).

Group Name: choose a name for the group you are creating.

Description: use this space to describe the groups’ interests.

Color: If you are creating more than one group, you can distinguish between them using different colors. Tags that were assigned by members of the group will show in view according to their color. This will assist you in identifying tags which were assigned to records by different groups’ members.

Invite members: separate each name by a semicolon or a new line. There is no limit on the number of members you can invite

When done, click on “create group”

You can cancel the creation of a group by using “cancel”

When a group is created, the next time you will tag a record, you will receive an option to tag to a group.

 

You can also and notify the group when new records are tagged.
