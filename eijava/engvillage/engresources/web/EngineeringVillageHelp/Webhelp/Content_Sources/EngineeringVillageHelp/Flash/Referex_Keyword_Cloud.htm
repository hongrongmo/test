Referex Keyword Cloud

The Referex Keyword Cloud feature displays which keywords appear in a particular book most frequently.  The Keyword Cloud offers visualization of word frequency within the book. The size of each term is determined with corresponding weights. Font sizes are used to indicate frequency of use.

When a user clicks on the “Book Details” link of a particular result, the Keyword Cloud will appear next to the Table of Contents.

Hovering over a keyword with the mouse shows how many times the keyword appears in that book.  

By clicking on the keyword, the user will see the pages on which the word is mentioned.
