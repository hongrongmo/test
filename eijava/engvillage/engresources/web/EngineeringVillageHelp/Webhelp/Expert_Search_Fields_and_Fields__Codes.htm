Expert Search Fields and Field Codes

In Expert search you can define the fields in which you would like the key words to be searched by using the field code.

Next to each field, in brackets, are the databases in which these fields are available:

c = Compendex

i = Inspec

n = NTIS

pc = PaperChem

cm = Chimica

cb = CBNB

el = EnCompassLIT

ep = EnCompassPAT

g = GEOBASE

f = GeoRef

u = US Patents

pa = Referex

Below are the fields, their corresponding codes, and description.
