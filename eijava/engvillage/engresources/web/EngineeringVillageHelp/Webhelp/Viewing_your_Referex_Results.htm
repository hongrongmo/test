Viewing your Referex Results

There are four options for viewing specific citations in your search results:

1. Page Details
2. Book Details
3. Read Page
4. Read Chapter

Page Details

Page details displays bibliographic information on the citation as well as a brief excerpt from the text with search terms highlighted. Hyperlinked elements facilitate easy navigation to other key information including authors, ISBN, Keywords, Subject Terms and Referex Collection. From the Page details screen you can navigate to Book Details, Read Page, Read Chapter, or Read Book.

 

 

Book Details

The Book Details view displays abbreviated bibliographic information on the title as well as an informative and detailed synopsis of the book contents. There is also a full table of contents for the book which is hyperlinked to book chapters. Clicking on any item in the table of contents brings up the full text of the relevant chapter. Hyperlinked elements within the Book Detail screen enable further flexible navigation and include author name, subject terms and collection. From the Book details page you can navigate to Page Details, Read Page, Read Chapter, or Read Book.

 

 

Read Page

Read page displays an enhanced PDF version of the page with multiple options for navigating through the rest of the chapter or book. Search terms are highlighted within the page; alternatively, highlighting can be turned off by clicking on the pen icon in the top of the display.

 

Read Chapter

Read Chapter opens an enhanced PDF file in a new window. It displays the full text of the chapter in which the citation occurs. You can read the chapter by scrolling through the text or you can print out the full chapter. Search term highlighting is not used on Read Chapter.

 

Read Book

The Read Book viewing option is accessible via the Page Details and Book Details views. It is not accessible from the Search Results page. Read Book will open a PDF file in a new window. You can scroll through the book using the Bookmarks on the left.
