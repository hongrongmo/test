Creating Groups

  1. Click on View/Edit groups.

  1. Click on Create a New Group (+).

  1. Group Name: choose a name for the group you are creating.

  2. Description: use this space to describe the groups’ interests.

  3. Color: If you are creating more than one group, you can distinguish between them using different colors. Tags that were assigned by members of the group will show in view according to their color. This will assist you in identifying tags which were assigned to records by different groups’ members.

  4. Invite members: separate each name by a semicolon or a new line. There is no limit on the number of members you can invite

  5. When done, click Create Group.

You can cancel the creation of a group by using “cancel.”

When a group is created, the next time you tag a record, you will receive an option to tag to a group.

 

You can also notify the group when new records are tagged.
