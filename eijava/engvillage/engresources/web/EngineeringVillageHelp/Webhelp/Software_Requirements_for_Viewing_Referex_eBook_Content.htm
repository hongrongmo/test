Software Requirements for Viewing Referex eBook Content

To read the full text of Referex pages, chapters or books, you will need to ensure that the following is installed and configured on your computer:

• Adobe Acrobat 5.0 or later (Reader or full version). Adobe 7.0 is recommended.

• Click here to see settings for the Adobe Acrobat Reader

If you need to install Adobe Acrobat, new versions and updates to previous versions can be downloaded from: http://www.adobe.com/products/acrobat/readstep2_allversions.html

Note: if this is not your personal computer, you may need a system administrator to install new software to this computer.
