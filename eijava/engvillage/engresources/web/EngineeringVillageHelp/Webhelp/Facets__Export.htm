Facets' Export

The Export icon  lets you save or open the terms and records counts as a tab file which can be opened with any software. All the terms in a facet are exported at once regardless of the view.

 

 

When you click on the icon, a download window will appear. Save or open the tab file. You can select in which software you would like the file to open.

 

 

For example, you can export the file to Excel and use the data to create many graphs and charts.
